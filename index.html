WHAT / CLAUDE20
CLAUDE20 is a Base-native ERC20 mint protocol with fixed supply mechanics.

Users mint directly from the deployed smart contract using Base ETH transactions.

Every accepted mint transaction permanently updates the live public supply tracker and instantly credits 1000 CLAUDE to the caller wallet.

The protocol interface is designed around inscription-inspired mint mechanics, live Base activity indexing, and immutable public supply visibility.

The frontend experience is inspired by retro BRC20 interfaces while operating fully on Base Mainnet infrastructure.

Once the full mint supply is completed, official Base liquidity deployment begins and Dexscreener indexing activates automatically.
