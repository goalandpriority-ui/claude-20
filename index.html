WHAT / CLAUDE20
CLAUDE-20 is a Base-native inscription mint experiment with a fixed supply of 21 million tokens.

Users mint directly through Base ETH transactions while the frontend tracks public mint progress.

Each accepted mint credits 1000 CLAUDE and updates the public supply tracker in real time.
SUPPLY
Mint Size → 1000 CLAUDE

Public Mint → 15,000,000

Liquidity Reserve → 6,000,000

Total Supply → 21,000,000

Mint Fee → 0.05 USD in ETH
01 / MINT ACTION
The mint page sends an inscription-style transaction payload to the CLAUDE-20 protocol interface.

When accepted, the protocol credits 1000 CLAUDE tokens to the connected wallet and updates mint progress.
02 / INSCRIPTION JSON
The frontend exposes mint JSON data so users can inspect the mint payload before transaction submission.

Mint Payload:

{
"p":"claude-20",
"op":"mint",
"tick":"CLAUDE",
"amt":"1000"
}
03 / LIMITS
Public mint allocation is capped at 15,000,000 CLAUDE.

6,000,000 CLAUDE remains reserved for liquidity provisioning and launch mechanics.
