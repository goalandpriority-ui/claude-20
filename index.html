CLAUDE20 OVERVIEW

CLAUDE-20 is a Base-native inscription mint protocol with a fixed supply of 21 million tokens.

All 21 million CLAUDE tokens are publicly mintable on Base network.

Mint transactions generate an on-chain JSON inscription which tracks mint progress.
