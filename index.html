WHAT / CLAUDE20
CLAUDE-20 is a Base-native inscription mint experiment with a fixed 21 million token supply. Users connect wallets, submit an inscription-style payload, and send a Base transaction directly to the treasury wallet. Mint progress is permanently tracked using Supabase indexing and reflected across the live activity feed, progress bar, and launch status interface.

01 / MINT ACTION

The CLAUDE-20 mint interface sends an on-chain Base ETH transaction directly to the treasury wallet. Once accepted, the frontend stores the mint action inside Supabase and updates the global protocol mint tracker.

Each inscription credits 1000 CLAUDE units and increments the public mint supply tracker.

{ "p":"claude-20", "op":"mint", "tick":"CLAUDE", "amt":"1000" }
02 / INSCRIPTION JSON

The frontend exposes both deploy and mint JSON payloads so users can inspect the raw inscription object before transaction submission.

The JSON itself acts as the protocol instruction layer and mirrors the behavior of inscription-based token standards.

03 / BASE LAYER

CLAUDE-20 operates entirely on Base Mainnet and uses standard wallet providers such as MetaMask, Coinbase Wallet, and Bitget Wallet.

The mint transaction itself is a native Base ETH transfer while protocol accounting is indexed through Supabase.
